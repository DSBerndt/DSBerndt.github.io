{{ "What is DFTBML?" }}
{{ "DFTBML is a quantum mechanical simulator. It is based on the Density Functional based Tight Binding (DFTB) model and uses novel machine learning techniques to lower computational cost and create accurate predictions. DFTBML stands for DFTB Machine Learning." }}
{{ "What Makes DFTBML Different from Other Machine Learning Models?" }}
{{ "DFTBML (DFTB Machine Learning) differs from other machine learning models in that it requires relatively little training data and does not function as a deep learning black box. Accuracy plateaus with around 20,000 training configurations and the resulting models are physically interpretable." }}
